# Installation

## Stable release

To install go2web, run this command in your terminal:

```sh
uv add go2web
```

Or if you prefer to use `pip`:

```sh
pip install go2web
```

## From source

The source files for go2web can be downloaded from the [Github repo](https://github.com/m33ga/go2web).

You can either clone the public repository:

```sh
git clone https://github.com/m33ga/go2web
```

Or download the [tarball](https://github.com/m33ga/go2web/tarball/main):

```sh
curl -OJL https://github.com/m33ga/go2web/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd go2web
uv sync
```
